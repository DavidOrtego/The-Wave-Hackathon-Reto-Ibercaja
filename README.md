# The-Wave-Hackathon-Reto-Ibercaja

Reto del hackathon de the wave 2026 basado en el reto propuesto por Ibercaja.

## 🚀 Descripción del Proyecto

Esta aplicación es una interfaz "mobile-first" enfocada en la sostenibilidad y el impacto social, diseñada como solución para el reto de Ibercaja en The Wave Hackathon 2026. 

El objetivo principal es permitir a los usuarios gestionar el impacto de sus finanzas, dándoles la posibilidad de destinar el "cashback" generado por sus compras a diferentes causas medioambientales y sociales.

## ✨ Características Principales

- **Dashboard de Impacto:** Los usuarios pueden visualizar su balance de cashback disponible para donar.
- **Selección de Causas:** Flujo intuitivo para que el usuario elija y asigne fondos a proyectos sociales o medioambientales específicos (ej. reforestación, limpieza de océanos, becas de estudio).
- **Seguimiento (Tracking):** Una línea de tiempo y gráficas de progreso que muestran el impacto real generado por las aportaciones del usuario en cada proyecto.
- **Diseño Móvil y Moderno:** Interfaz limpia, premium y accesible diseñada específicamente para dispositivos móviles utilizando los colores corporativos y la estética visual de Ibercaja.

## 💻 Tecnologías Utilizadas

Este proyecto ha sido desarrollado utilizando tecnologías nativas de la web para garantizar el máximo rendimiento y simplicidad sin depender de frameworks complejos:

- **HTML5:** Estructura semántica de la aplicación.
- **CSS3 (Vanilla):** Estilos personalizados, animaciones asombrosas y diseño responsivo "mobile-first".
- **JavaScript (Vanilla):** Lógica de la aplicación, interactividad, navegación entre vistas y actualización dinámica del DOM.

## 📁 Estructura del Proyecto

El repositorio consta de una estructura simple y directa:

```text
/
├── index.html   # Archivo principal con la estructura de todas las vistas
├── styles.css   # Hoja de estilos con el diseño visual y variables de la marca
├── app.js       # Lógica encargada del ruteo, estado e interactividad
└── README.md    # Este archivo que estás leyendo
```

## 🛠️ Cómo ejecutar el proyecto en tu entorno local

Para probar la aplicación en tu ordenador, no es necesario instalar ninguna dependencia:

1. Clona este repositorio o descarga el código fuente.
2. Abre la carpeta del proyecto.
3. Puedes abrir el archivo `index.html` directamente en tu navegador web de preferencia haciendo doble clic sobre él.
4. (Recomendado) Para una mejor experiencia (y evitar problemas menores de CORS con módulos JS locales si los hubiere), utiliza un servidor local estático como la extensión "Live Server" en VSCode o ejecuta:
   ```bash
   npx serve .
   ```
5. ¡Listo! Inspecciona la página con las herramientas de desarrollador y activa la vista de dispositivo móvil para ver el diseño correctamente.
